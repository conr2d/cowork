# Icons

These are **placeholder** icons (a solid indigo square) generated during the WP0
scaffold so the Tauri config references real files and the bundle builds.

Replace them with the real Cowork brand mark before any release: drop a square
source PNG (≥ 1024×1024) and run

```sh
pnpm tauri icon path/to/cowork-logo.png
```

which regenerates the full platform icon set (`.png`, `.ico`, `.icns`) in place.
