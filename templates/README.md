# Templates

Workspace templates shipped with Cowork.

**v0.1:** only a minimal **default workspace** — the contents seeded into
`~/cowork/` inside the guest during toolchain bootstrap (WP6), the directory the
user lands in at the end of setup. It is intentionally near-empty in v0.1.

The template **sharing/marketplace** system (and per-template locale maps) is a
v0.4 (Community) concern — out of scope here.
